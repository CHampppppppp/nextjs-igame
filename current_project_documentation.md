# iGame Lab 当前项目文档

## 项目概述

**项目名称**: iGame Lab (Intelligent Visual Modeling & Simulation Lab)  
**项目类型**: 学术实验室网站  
**技术栈**: HTML/CSS/JavaScript + Bootstrap + jQuery + PHP  
**当前状态**: 静态网站 + 邮件发送功能  
**实验室定位**: 杭州电子科技大学计算机学院图形图像所智能可视建模与仿真实验室

## 页面功能详解

### 1. 主页 (index.html)

### 3. 研究页面 (research.html)

#### 研究方向展示
实验室三个主要研究方向：

**3.1 网格生成 (Mesh Generation)**
- 研究重点：六面体网格生成、结构化网格自动化
- 应用领域：CAD/CAE工业软件、国防基础科研
- 相关论文：5篇核心论文，涉及边界元、快速多极方法等

**3.2 等几何分析 (Isogeometric Analysis)**
- 研究重点：计算域参数化、高精度仿真分析
- 核心技术：变分调和映射、拓扑一致参数化
- 学术成果：3篇ESI热点论文，30+篇SCI论文
- 应用价值：智能制造、虚拟现实高精度仿真

**3.3 视觉与学习 (Vision & Learning)**
- 研究重点：深度学习在计算机视觉中的应用
- 核心项目：
  - 智能计算艺术 (AiART)
  - 绘画机器人
  - 艺术风格迁移
  - 虚拟试衣
- 代表性成果：TIP、TCYB期刊发表，IROS会议论文

#### 技术特色
- 每个研究方向配有相关技术图片
- 论文列表包含完整引用信息
- 外部链接指向论文原文和项目演示

- **标题字体**: Merriweather系列 (merriweather_light, merriweatherbold, merriweatherbold_italic)

- **容器宽度**: Bootstrap标准容器 (最大1200px)

## 项目优势与局限性

### 优势
- **轻量级**: 无需服务器端运行环境
- **快速加载**: 静态资源，加载速度快
- **易部署**: 只需要Web服务器即可部署
- **维护简单**: 技术栈成熟，维护成本低

### 局限性
- **内容更新困难**: 需要技术人员手动修改HTML
- **功能扩展受限**: 静态网站无法实现复杂交互
- **搜索引擎优化**: 动态内容生成能力弱
- **用户互动不足**: 缺乏实时交互和个性化功能
