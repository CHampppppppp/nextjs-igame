# iGame Lab 网站页面结构文档

## 网站概述
iGame Lab（智能可视建模与仿真实验室）网站是一个学术实验室网站，采用HTML/CSS/JavaScript构建，包含响应式设计和现代化的用户界面。

## 网站主要页面

### 1. 主页 (index.html)
**页面位置**: `html/index.html`

#### 页面结构：
- **网站头部**
  - 实验室logo（左上角）
  - 杭州电子科技大学logo（右上角）
  - 导航菜单：主页、团队、研究、团建、联系我们

- **主轮播区域**
  - "欢迎来到iGame Lab" slide
  - "教师团队" slide

- **实验室简介区域**
  - 实验室基本信息介绍
  - 团队成员构成
  - 研究方向概述
  - 学术成果统计

- **近期论文区域**
  - 展示3篇最新学术论文
  - 包含论文标题、作者、期刊信息

- **最新信息区域**
  - 实验室最新动态
  - 论文发表、人员加入、项目获批等消息

- **欢迎加入我们区域**
  - 研究生招生信息
  - 联系方式链接

- **团队建设区域**
  - 展示5张团队活动图片
  - 包含活动时间和标题

- **页脚区域**
  - 实验室地址信息
  - 友情链接（学校官网、学院官网等）
  - 微信公众号二维码

### 2. 团队页面 (team.html)
**页面位置**: `html/team.html`

#### 页面结构：
- **页面标题**: "团队成员"

- **教师团队区域**
  - 展示7位教师成员的卡片
  - 每个卡片包含：
    - 教师照片
    - 姓名和职称
    - 链接到个人主页

- **学生团队区域**
  - **博士研究生**：4名，按年级分组
  - **硕士研究生**：按年级分组（2023级、2024级、2025级）
  - 总计约60+名研究生

### 3. 研究页面 (research.html)
**页面位置**: `html/research.html`

#### 页面结构：
- **页面标题**: "研究"

- **研究课题区域**
  - **网格生成**：六面体网格、结构化网格、工业软件应用
  - **等几何分析**：计算域参数化、高精度仿真、拓扑一致性
  - **视觉与学习**：深度学习、计算机视觉、艺术机器人

每个研究方向包含：
- 研究内容描述
- 相关论文列表
- 研究成果说明

### 4. 团队建设页面 (teambuilding.html)
**页面位置**: `html/teambuilding.html`

#### 页面结构：
- **页面标题**: "团队建设"

- **活动图片展示区域**
  - 6张团队活动照片网格布局
  - 活动内容包括：
    - 学术报告
    - 年会报告
    - 新生聚餐

    - 19年会报告
    - 绍兴团建大禹陵合影

### 5. 联系我们页面 (contact.html)
**页面位置**: `html/contact.html`

## 技术架构

### 前端技术栈：
- HTML5/CSS3
- JavaScript (jQuery, Bootstrap)
- Vue.js (可视化页面)
- 响应式设计

### 样式和资源：
- Bootstrap CSS框架
- Font Awesome图标库
- Bootstrap Icons
- 自定义CSS样式
- 图片资源（团队照片、活动照片、研究图片）

### 页面布局特点：
- 响应式网格布局
- 卡片式设计
- 轮播图组件
- 粘性导航栏
- 统一的页脚设计

## 网站特色功能

1. **响应式设计**：适配不同设备屏幕
2. **多级导航**：清晰的页面层次结构
3. **学术展示**：突出研究成果和团队实力
4. **招聘平台**：提供教师和学生招聘信息
5. **团队展示**：完整的教师和学生团队介绍
6. **活动记录**：团队建设活动图片展示

这个网站完整地展示了iGame Lab实验室的学术实力、团队构成、研究方向和招生信息，为访问者提供了全面的实验室概况。
